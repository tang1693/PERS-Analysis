# PERS - Photogrammetric Engineering & Remote Sensing 论文数据集

## 项目概览

本项目包含来自 ASPRS (American Society for Photogrammetry and Remote Sensing) 期刊的学术论文数据集。

## 数据集信息

- **文件名**: `ALL_articles_Update_cleaned.csv`
- **论文数量**: 1,582 篇
- **时间范围**: 2024-2026年
- **字段结构**:
  - Title (标题)
  - Authors (作者)
  - Pages (页码范围)
  - Access (访问类型: Open Access / Subscribed content)
  - URL (论文链接)
  - Abstract (摘要)

## 研究领域覆盖

根据初步分析，数据集涵盖以下主要研究方向：

### 🛰️ 遥感技术
- 高光谱遥感
- SAR (合成孔径雷达)
- LiDAR 点云处理
- 卫星影像分析

### 🤖 深度学习应用
- 目标检测 (YOLO, CNN)
- 语义分割 (U-Net, Transformer)
- 图像分类
- 点云处理

### 🌍 应用领域
- 土地利用/土地覆盖分类
- 建筑物提取
- 农作物分类
- 森林监测
- 灾害评估
- 城市规划

### 📊 热点技术
- Transformer 架构
- 多模态数据融合
- 时序分析
- 小样本学习
- 自监督学习
